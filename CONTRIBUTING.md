# Contribuir a OSINT Deck

Gracias por tu interés en contribuir a OSINT Deck.  
Las contribuciones son bienvenidas siempre que mantengan la estabilidad, seguridad y calidad del proyecto.

---

## Formas de contribuir
- Reportar errores
- Proponer nuevas herramientas OSINT
- Enviar mejoras de código o documentación
- Crear nuevos templates JSON
- Discutir funcionalidades en Issues

---

## Reglas básicas
1. Las contribuciones deben estar asociadas a un Issue abierto (o abrir uno nuevo).
2. Los cambios deben enviarse en un Pull Request con una descripción clara.
3. Mantener el estilo de código:  
   - PHP >= 8  
   - WordPress Coding Standards (WPCS) cuando sea posible  
4. No enviar código con dependencias externas no revisadas.
5. Evitar incluir credenciales, claves o endpoints privados.
